# Student Time Management App

A modern, responsive **Student Time Management App** built with **HTML, CSS, and JavaScript** to help students organize their tasks, assignments, quizzes, project ideas, and job applications in one place. The app features a clean, dark-themed design and works smoothly on desktop, tablet, and mobile devices.

---

## Features

- **Task Management**
  - Add, edit, mark as done, and delete tasks.
  - Persistent storage using `localStorage`.

- **Assignment Tracking**
  - Add and view assignments with subject, date, and description.

- **Quiz Planner**
  - Add and manage quizzes with subject, date, and details.

- **Project Ideas**
  - Keep track of project ideas along with descriptions.

- **Job Applications**
  - Manage company applications, job titles, and details.

- **Responsive Design**
  - Fully responsive layout for **desktop**, **tablet**, and **mobile** screens.
  - Sidebar stays fixed on large screens for easy navigation.

- **Local Storage**
  - All data persists across sessions using the browserâ€™s `localStorage`.
